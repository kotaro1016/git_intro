GitとGithubの使い方

1。対象ファイルを更新

2．ターミナル起動

3．ステージングに追加
　git add .
　git add "ファイル名"

4．ローカルリポジトリに追加
　git commit -v  //-v をつけると差分が表示される。

5．vimが起動するのでコメントを記載

　iを押下し、入力モードにはいる。

　1行目：変更内容
　2行目：空行
　3行目：変更理由

　入力したらesc->:qw

6．Githubへpush
　git push -u origin "brunch名"
　git push -u origin main		//このファイルが保存されている先のブランチ名はmain

■ファイルの削除

1．ファイルを削除
　rm "ファイル名"		//ターミナルから削除する場合

2．ステージング環境から削除
　git rm "ファイル名"
　git commit "ファイル名"
